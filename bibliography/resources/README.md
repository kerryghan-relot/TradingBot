# Ressources

Ce dossier rassemble les copies locales des ressources les plus pertinentes citées dans [`../bibliography.md`](../bibliography.md). Toutes n'y figurent pas : les ouvrages sous droits d'auteur en sont exclus, et seules les ressources librement et légalement disponibles (articles en libre accès, préprints, documentation publique) ont vocation à y être déposées.

Nommez chaque fichier d'après le numéro de sa référence dans la bibliographie, suivi d'un court intitulé — par exemple `13-kalman-1960.pdf` pour la référence 13.
